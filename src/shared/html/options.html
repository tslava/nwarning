Environment URL Pairs

Configure pairs of production and development URLs. The extension will show environment warnings when visiting these URLs.

You can use wildcards (*) to match dynamic parts of domains. For example:

  • Production: *.production.example.com
  • Development: *.dev.example.com
  • Or more specific:
  • Production: example.com
  • Development: dev.example.com

When switching environments, the extension will preserve the dynamic part that matches the wildcard (*).

Local Storage Variables Tracking

Monitor specific localStorage keys to display warnings when their values indicate a production configuration.

The extension will:

  • Show warning text when the key is found in localStorage

Example keys to track:

  • bestcompany-production-enabled
  • use-prod-database
